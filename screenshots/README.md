# Screenshots

This directory contains screenshots and GIFs for the project README.

## How to Add Screenshots

1. Run the development server: `npm run dev`
2. Open `http://localhost:3000` in your browser
3. Take screenshots of:
   - Main gameplay view (character roster + action area + mission log)
   - Mission log with history entries
   - Win/lose screens
4. Save images to this directory
5. Update the README.md to reference the actual screenshot files

## Placeholder Files
